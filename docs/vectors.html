1.4 Vectors

Objecten kan je ook samenvoegen in een vector:

p1 <- c(naam, woonplaats, leeftijd)

Je kan dan je naam, woonplaats of leeftijd aanroepen door respectievelijk p1[1], p1[2], p1[3] als opdracht te geven. De vector p1 is zelf ook een object, die bestaat uit de waarden van de andere drie objecten. Vectoren worden veelvuldig gebruikt in R. Als ze alleen getallen bevatten kan je er ook berekenen mee uitvoeren.
